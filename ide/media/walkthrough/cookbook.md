The **Cookbook** is a searchable gallery of solved HL7 routing/transform problems — crosswalking a
code, splitting a batch by OBR, enriching via a live database lookup, fanning a message out to
several outbounds, and more. Each entry inserts real, editable Python at your cursor; nothing here
generates code from a form — it's a static index of working examples you adapt by hand.

Search for a problem you recognize (e.g. "split", "crosswalk", "fan-out"), then insert and edit.
